## Pedalea. Genera energía. Transforma tu entorno.  

**Eco-Urban Power** es una iniciativa sostenible que convierte el ejercicio físico en energía limpia. Diseñada para **gimnasios, universidades, hoteles y espacios deportivos de Cartagena, Colombia**, esta solución transforma bicicletas estáticas en generadores eléctricos que reducen costos energéticos y promueven la conciencia ambiental.

## ¿Cómo funciona?
1. El usuario pedalea en una bicicleta equipada con el sistema **Eco-Urban Power**.  
2. Un **generador** convierte la energía cinética del pedaleo en electricidad.  
3. La energía se **almacena** en baterías o se **integra directamente a la red eléctrica** del lugar.  
4. El usuario puede **visualizar en tiempo real** la energía generada mediante la aplicación web.

## Módulo: Diseño del sistema
El diseño de **Eco-Urban Power** se centra en la integración funcional y visual de la generación de energía sostenible.  

- **Interfaz limpia y moderna:** basada en componentes modulares y un esquema de color verde-gris que simboliza sostenibilidad y tecnología.  
- **Arquitectura eficiente:** el módulo de visualización conecta los datos del generador con la app mediante gráficos dinámicos.  
- **Accesibilidad:** diseñado con prioridad para compatibilidad móvil y una experiencia fluida para todo tipo de usuario.  
- **Optimización energética:** los componentes electrónicos se seleccionaron para maximizar la conversión y minimizar pérdidas.
